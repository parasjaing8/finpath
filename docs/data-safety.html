Data Safety – Play Console Reference

Last updated: April 11, 2026  ·  App: FinPath (com.anonymous.finpath)

This page provides exact answers for the Play Console → App content → Data safety form. Fill the form exactly as described below. Do not publish this page publicly as a user-facing document — use privacy-policy.html for that.

Section 1 – Data Collection and Security

Does your app collect or share any of the required user data types?

Yes — the app collects personal info, financial info, and app activity data.

Is all of the user data collected by your app encrypted in transit?

Yes — all network connections use HTTPS (Google Play Billing only). No app data is transmitted to our own servers.

Do you provide a way for users to request that their data is deleted?

Yes — users can delete a profile and all associated data from within the app (Profile tab → Delete Profile). Uninstalling the app removes all locally stored data.
